# Manual de imágenes - Refugio La Arboleda

Las imágenes finales de la página pública están en:

`public/imagenes/refugio`

Para cambiar una imagen sin tocar código:

1. Abrir la carpeta `public/imagenes/refugio`.
2. Reemplazar la imagen que quieras cambiar conservando exactamente el mismo nombre.
3. Guardar los cambios.
4. Hacer commit y deploy.

## Imágenes principales

- `logo-refugio.png`: logo usado en navbar y footer.
- `hero-cabana-exterior.jpg`: imagen principal del Hero.
- `og-refugio-la-arboleda.jpg`: imagen que se comparte en WhatsApp, Facebook y otras redes cuando se envía el enlace.

## Nota importante

Para que la dueña pueda subir imágenes directamente desde el panel, se necesitaría una mejora futura con Supabase Storage y módulo de galería administrable. Por ahora, este sistema es simple y estable: reemplazar el archivo conservando el mismo nombre actualiza la imagen en la web.
